# 💰 Expense Tracker

Personal finance tracking web application built with ASP.NET Core MVC.

## 🚀 Features

- Add income and expenses
- Filter transactions by date, category and type
- Pagination
- Monthly reports
- Category management
- User authentication

## 🛠️ Technologies

- ASP.NET Core MVC
- Entity Framework Core
- SQL Server (LocalDB)
- Bootstrap

## 📊 Description

This application allows users to track their personal finances by managing income and expenses.  
It includes filtering, reporting, and categorization features to give a clear overview of financial activity.

## ⚙️ How to run

1. Clone the repository
2. Open the solution in Visual Studio
3. Run the application
